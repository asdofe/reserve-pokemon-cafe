# Installation
1. 打開Chrome，chrome://extensions
2. 打開開發人員模式
3. 把下載下來的zip解壓縮，把解壓縮資料夾的路徑載入未封裝檔案(manifest.json同一層)
4. 去到https://reserve.pokemon-cafe.jp/，右下角有一個開始的按鈕按下去即可自動開始點擊 (這個預約網站很容易鎖IP自己謹慎使用)
